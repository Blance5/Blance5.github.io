StormSim.AI

KnightHacks Hackathon 2023 Category Winner

With a team of 3 others I had never met before, we were able to develop a fullstack application that allows users to model damages of a potential hurricane. I'm from Florida, and we know what to expect when a hurricane hits, but there are many others who are unaware of the possible effects of a hurricane, this is why we wanted to create an app that allows people to compare the effects of hurricanes they create, to obtain a better understanding of what may happen if a hurricane strikes. The application allows users to hand-draw the path of a potential hurricane, and select their location of residence. Using propritary algorithms and the Google Map Visualization library, we are able to provide meaningful statistics and visualizations to the user displaying how the hurricane may affect them. This project was the winner of the Knight Hacks 2023 Best Use of Google Cloud category

The following technologies were used in the making of this project

  • Google Maps API
  • JavaScript/HTML/CSS
  • Google Maps Visualization/Places/Drawing Libraries
  • Node.js
  • Google Maps Heatmap
  • ChatGPT Integration to query hurricane statistics
